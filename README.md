# 💳 Payment Card Fraud Detection 2025

## 📊 Project Overview

This project analyzes customer transaction data to identify **spending patterns, product performance, and fraud detection signals** using SQL.

---

## 🗂 Dataset Summary

* Total Transactions: **2,133**
* Unique Customers: **2,133**
* Each customer has only one transaction

---

## 🧠 Key Insights

* Total Revenue: **372,451.82**
* Average Transaction Value: **174.61**
* Fraud Rate: **3.09%**
* Fraud peaks during early morning hours
* Top stores with highest fraud:

  * POPUP-MILAN
  * FLAGSHIP-PARIS

---

## 🔍 SQL Skills Used

* Aggregations (`SUM`, `AVG`, `COUNT`)
* Window Functions (`LAG`, `DENSE_RANK`)
* Case Statements
* Fraud Detection Techniques
